# Synapse Rescue

Razer Profile Manager. Backup and restore Razer Synapse profiles.

![Screenshot](screenshot.png)

## What is Synapse Rescue?

Synapse Rescue is a free Windows utility that backs up, restores, and converts your Razer Synapse profiles. Never lose your custom lighting, macro, and keybinding configurations again — export them to portable files and restore with one click.

## Features

- Backup all Razer Synapse profiles to portable files
- One-click restore from backup
- Export profiles before Windows reinstalls or hardware changes
- Convert profiles between Synapse 2 and Synapse 3 formats
- View profile details without launching Synapse
